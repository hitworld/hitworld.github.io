概览

CSRF攻击的本质是攻击者能知道某个操作的所有参数,通过构造url让受害者访问从而伪造请求.

使用

CSRF常使用标签的src跳转,如:<img>,<script>,<iframe>.

或者使用JS方法等等.

JSON Hijacking

JSON是JavaScript的子集,一种轻量化的数据交换格式,实际上就是键值对.

JSON Hijacking实际上就是利用CSRF漏洞盗取用户在漏洞网站的铭感信息.
