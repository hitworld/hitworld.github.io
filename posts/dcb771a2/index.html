w4rd3n-win环境搭建

发表于 2017-09-01 | 分类于 w4rd3n | 热度： ℃
本文字数： 3.9k | 阅读时长 ≈ 4 分钟

基本环境

Java

JDK:Java开发环境,其中包含运行环境.

Tomcat:Java服务端开发环境,例如:Servlet,JSP等.

python

最好有python2和python3,使用virtualenvwrapper(Python3,不支持powershell)管理.

数据库

MySQL Installer for Windows.用于管理Windows下MySQL环境.

虚拟机

VMware:虚拟机管理软件.

Node.js

基于Chrome V8引擎的JavaScript运行时环境.

PHP

PHP运行时环境,Apache24,PHP7.

需要修改Apache24\conf\httpd.conf.

web安全

dirsearch

web目录扫描.https://github.com/maurosoria/dirsearch.基于python3.

sqlmap

开源渗透测试工具,自动检测和利用SQL注入漏洞.http://sqlmap.org/.

metasploit

渗透测试框架.https://www.metasploit.com/.

AntSword

可视化网站管理工具,常用于连接后门.https://github.com/AntSwordProject/antSword/releases.

BurpSuite Pro

web渗透测试工具.

Fiddler

适用于任何浏览器,系统或平台的免费Web调试代理工具.https://www.telerik.com/fiddler.

d_safe

web后门检测工具.http://www.d99net.net/.

Nmap

网络扫描工具.https://nmap.org/.

wireshark

网络协议分析器.https://www.wireshark.org/.

Acunetix

web自动化漏洞检测与渗透测试工具.

取证分析

010 Editor

十六进制编辑器,可以通过模板分析文件.

volatility

先进的内存取证框架.https://github.com/volatilityfoundation/volatility.

二进制工具

dex2jar

Android逆向工具.https://github.com/pxb1988/dex2jar.

apktool

Android逆向工具.https://ibotpeaches.github.io/Apktool/.

dnSpy

.NET调试器和程序集编辑器.https://github.com/0xd4d/dnSpy.

ILSpy

.NET反编译器.https://github.com/icsharpcode/ILSpy.

petools

便携式PE操纵工具.https://github.com/petoolse/petools.

PEiD & Exeinfo PE

检测常见的各种壳.

jd-gui

独立的Java Decompiler GUI.https://github.com/java-decompiler/jd-gui.

JEB

反编译和调试二进制代码.

OllyDbg

Windows平台的32位分析调式工具.http://www.ollydbg.de/.

IDA

二进制研究者必备的逆向工具.

upx

可执行文件的加壳工具.https://github.com/upx/upx.

VB Decompiler Pro

用来反编译VB编写的程序的工具.

winchecksec

Windows平台的保护机制检测工具.https://github.com/trailofbits/winchecksec.

pwintools

Windows平台pwntools,存在bug.https://github.com/masthoon/pwintools.

AppJailLauncher

Windows二进制程序挂起监听工具.https://github.com/trailofbits/AppJailLauncher.

huorong

安全软件.https://www.huorong.cn/.

x64dbg

Windows平台的32/64位分析调式工具.https://x64dbg.com/#start.

日常工具

7-Zip

文件存档器.https://www.7-zip.org/.

hyper

终端.https://hyper.is/.

cmder

终端.

Everything

基于名称快速定位文件和文件夹的工具.https://www.voidtools.com/zh-cn/.

Wox

快速启动器,需要Everything.http://www.wox.one/.

pandownload

用于快速下载百度云文件.https://pandownload.com/.

PotPlayer

视频播放器.https://potplayer.daum.net/.

Hexo

基于Nodejs,快速,简洁且高效的博客框架.https://hexo.io/zh-cn/docs/index.html.

ShadowsocksR

小飞机.https://github.com/shadowsocksrr/shadowsocksr-csharp/releases.

Typora

markdown编辑器.https://www.typora.io/.

Rufus

光盘映像文件制作.https://rufus.ie/.

UltraISO

光盘映像文件制作/编辑/转换工具.https://cn.ultraiso.net/xiazai.html.

Xftp/Xshell

https://www.netsarang.com/zh/xftp/.

https://www.netsarang.com/zh/xshell/.

Acrobat DC

PDF编辑器.

TIM

办公版QQ.https://tim.qq.com/.

WeChat

微信.https://pc.weixin.qq.com/.

百度网盘

网盘.

qBittorrent

P2P协议下载器.https://www.qbittorrent.org/.

QQ音乐

音乐播放器.https://y.qq.com/.

MindMaster

思维导图编辑器.https://mm.edrawsoft.cn/.

Chrome

Google浏览器.https://www.google.com/intl/zh-CN/chrome/.

Firefox

火狐浏览器.https://www.mozilla.org/zh-CN/firefox/new/.

Office

微软办公软件.

开发

Git

免费的开源分布式版本控制系统.https://git-scm.com/.

Android Studio

Google官方IDE.https://developer.android.com/studio/.

加快gradle的build.当build卡在下载环节时.查看<your project name>\gradle\wrapper\gradle-wrapper.properties.distributionUrl的值就是下载的url,访问这个url下载这个包.

C:\Users\<your user name>\.gradle\wrapper\dists\目录下有和你这个包名字一样的文件夹,进去发现只有个名字奇怪的文件夹,将里面的东西删除,将下载的包放进去,重新打开Android Studio即可.

IntelliJ IDEA

Java开发IDE.https://www.jetbrains.com/idea/.

VS Code

https://code.visualstudio.com/.

Qt

C++跨平台的软件开发工具包.https://www.qt.io/cn.

WebStorm

前端IDE.https://www.jetbrains.com/webstorm/.

Visual Studio

微软官方IDE.https://visualstudio.microsoft.com/zh-hans/.

Navicat Premium 12

数据库开发工具.

nasm

AT&T汇编器.https://www.nasm.us/pub/nasm/releasebuilds/.

smali

smali/baksmali.https://github.com/JesusFreke/smali.
